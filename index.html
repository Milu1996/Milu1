Tower Game

Build the tallest tower by stacking blocks with perfect precision. How high can you go?

关于 Tower Game

Tower Game 是一款令人上瘾的堆叠游戏，你需要将方块精准地叠加，挑战最高的高楼。考验你的手速与精准度！

游戏特色

  • 简单但极具挑战性的玩法
  • 精准的方块堆叠机制
  • 难度逐步提升
  • 分数记录系统
  • 完美堆叠奖励

玩法说明

点击或轻触屏幕即可下落方块，目标是让方块尽量完美地叠加在一起。每次成功堆叠可获得分数，连续完美堆叠有额外奖励。随着高楼升高，游戏难度也会增加！

游戏规则

  • 初始拥有 3 点生命值（HP）
  • 每次下落方块消耗 1 点 HP
  • 成功堆叠得 25 分
  • 完美堆叠得 50 分，连续完美有额外加分
  • HP 归零游戏结束

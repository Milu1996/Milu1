About Tower Game

Tower Game is an addictive stacking game where you need to stack blocks with precision and challenge the highest tower. Test your speed and accuracy!

Game Features

  • Simple yet highly challenging gameplay
  • Precision block stacking mechanics
  • Progressively increasing difficulty
  • Score tracking system
  • Perfect stack bonuses

How to Play

Click or tap to drop the block. The goal is to stack the blocks as perfectly as possible. Each successful stack earns you points, and consecutive perfect stacks grant extra bonuses. The game gets harder as your tower grows taller!

Game Rules

  • Start with 3 HP (lives)
  • Each block drop costs 1 HP
  • Successful stack: +25 points
  • Perfect stack: +50 points, extra bonus for consecutive perfects
  • Game over when HP reaches zero
